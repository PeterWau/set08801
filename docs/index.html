How to use Povo Golf

Dashboard

Our Dashboard will guide your through your learning journey.

Add training videos

  1. Go to Training Videos
  2. Click Add Video
  3. Enter:
    • URL e.g. a YouTube play link
    • Coach e.g. Danny Maude
    • Select a Skill Focus - Pick one:

  4. Click Save Video

Record practice session

  1. Go to Record Practice Session
  2. Select a Club - Pick one:
  3. Select a Skill Focus - Pick one:

  4. Add shots:
    • Enter a Distance Estimate
    • Select a Shape - Pick one:

  5. Click Save Session

Check your club distances

  1. Go to Distance Report
  2. It shows a breakdown of average club distance that you can take to the course.
